# Contributing

1. Keep MiniMax App, OpenCode Zen, and OpenCode Go as strict source boundaries.
2. Never add Codex or non-MiniMax model data to this application.
3. Add or update tests for changes to aggregation, filtering, dates, or pricing.
4. Run `npm run check` and `npm test` before opening a pull request.
5. Do not commit databases, logs, API keys, generated runtimes, or user paths.
